**#JeevanRaksha – Smart Emergency Assistance Platform**

JeevanRaksha is a web-based emergency assistance platform designed to provide instant support during critical situations. The project focuses on reducing response time, improving location awareness, and guiding users with first-aid instructions to ensure faster and safer medical help.

This application acts as a digital life-saving companion, enabling users to quickly access emergency tools such as live location sharing, nearby hospital search, traffic navigation, and instant medical guidance.

 **Problem Statement**

1. During emergencies, people often face:

2. Delay in contacting help

3.Difficulty finding the nearest hospital

4.Lack of first-aid knowledge

5.Poor traffic awareness

6.Inability to share accurate location quickly

JeevanRaksha addresses these challenges by providing a single interactive dashboard for emergency response and coordination.

**Solution**

JeevanRaksha integrates location services, navigation tools, and medical guidance into a simple, user-friendly interface that works directly from a browser without installation.

 **Features**

1.SOS Alert – Instantly triggers an emergency alert message.

2.Live Location Map – Opens the user’s current GPS location in Google Maps.

3.Find Nearby Hospitals – Automatically searches for hospitals near the user.

4.First Aid Guidance – Provides quick medical instructions (CPR, bleeding control, choking help).

5.Alert Family – Shares live location through WhatsApp link.

6.Traffic View – Displays real-time traffic conditions on Google Maps.

7.Chat Help Assistant – Simple chatbot for first-aid tips.

8.Responsive UI – Clean, modern, and mobile-friendly interface.

**Tech Stack**

1.HTML5 – Structure and content

2.CSS3 – Styling, animations, responsive design

3.JavaScript – Interactivity and location features

4.Google Maps Integration – Navigation, traffic, and hospital search

5.Geolocation API – Live user location detection

**Objectives**

1.Reduce emergency response time

2.Improve access to nearby medical facilities

3.Provide immediate first-aid guidance

4.Enable fast and accurate location sharing

5.Create a simple and accessible emergency dashboard

**Future Enhancements**

1.AI-powered medical chatbot

2.Ambulance tracking system

3.Multi-language support

4.Voice-activated SOS

5.Mobile app version

6.Hospital bed & blood bank availability integration

**Conclusion**

JeevanRaksha demonstrates how web technology can be used to build a practical, life-saving emergency solution. By combining real-time location services, navigation tools, and first-aid assistance, the platform aims to make emergency support faster, smarter, and more accessible for everyone.
